# 🔐 Projet Spring Security avec Authentification JPA

Ce projet implémente un système d'authentification robuste utilisant **Spring Boot**, **Spring Security**, et **JPA/Hibernate** avec une base de données **MySQL**.

## 🚀 Technologies Utilisées
* **Backend** : Java 17+, Spring Boot 3, Spring Security.
* **Frontend** : Thymeleaf, CSS3, Google Fonts (Poppins).
* **BDD** : MySQL, Hibernate (JPA).
